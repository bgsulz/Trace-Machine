Scans embedded EXIF metadata and surfaces any tags or software identifiers that commonly appear in AI image generators or heavy post-processing workflows. EXIF (Exchangeable Image File Format) is a standard for storing technical information inside image files—originally designed for cameras to record settings like shutter speed and lens type, but now widely used by software to embed their own markers and parameters.

AI image generators like Stable Diffusion (through interfaces like Automatic1111 and ComfyUI) often write detailed generation parameters into EXIF fields, including the prompt used, model version, sampling steps, and seed values. Photo editing software may also leave telltale signatures. When present, these markers can provide strong clues about an image's synthetic or heavily edited origins.

Important limitation: EXIF data can be freely edited or completely stripped using basic tools, so its presence is informative but its absence proves nothing. A sophisticated actor can easily remove or falsify these markers.
